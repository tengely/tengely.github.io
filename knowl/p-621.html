Now, consider the sum of the squares of the numbers in a row. We can find a pattern here, as well:

\begin{align*} 1^2 \amp = 1,\\ 1^2 + 1^2 \amp = 2,\\ 1^2 + 2^2 + 1^2 \amp = 6,\\ 1^2 + 3^2 + 3^2 + 1^2 \amp = 20,\\ 1^2 + 4^2 + 6^2 + 4^2 + 1^2 \amp = 70,\\ 1^2 + 5^2 + 10^2 + 10^2 + 5^2 + 1^2 \amp = 252,\\ 1^2 + 6^2 + 15^2 + 20^2 + 15^2 + 6^2 + 1^2 \amp = 924\text{.} \end{align*} in-context
