We show another proof here, which is usually attributed to Carl Friedrich Gauss. 1  Let \(S\) be the sum of the integers from 1 to \(n\text{,}\) and write the same sum in reverse order below:

\begin{align*} S \amp = 1 + 2 + \dots + (n-1) + n,\\ S \amp = n + (n-1) + \dots + 2 + 1.\\ \end{align*}

Adding the two equations together, we obtain

\begin{align*} 2S \amp = (1+n) + (2 + n-1) + \dots + (n-1 + 2) + (n+1),\\ 2S \amp = (n+1) + (n+1) + \dots + (n+1) + (n+1),\\ 2S \amp = n \cdot (n+1),\\ S \amp = \frac{n\cdot (n+1)}{2}\text{.} \end{align*} in-context
