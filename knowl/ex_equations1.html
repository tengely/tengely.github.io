Exercise2.7.14

How many integer solutions do the following equations have?

  1. \(x + y + z = 9\text{,}\) where \(x\geq 1\text{,}\) \(y\geq 1\text{,}\) \(z \geq 1\text{;}\)

  2. \(x + y + z = 8\text{,}\) where \(x\geq 0\text{,}\) \(y\geq 0\text{,}\) \(z \geq 0\text{;}\)

  3. \(x + y + z = 7\text{,}\) where \(x\geq 0\text{,}\) \(y\geq 0\text{,}\) \(z \geq 0\text{;}\)

  4. \(x + y + z = 11\text{,}\) where \(x\geq 2\text{,}\) \(y\geq 2\text{,}\) \(z \geq 2\text{;}\)

  5. \(w + x + y + z = 9\text{,}\) where \(w \geq 1\text{,}\) \(x\geq 1\text{,}\) \(y\geq 1\text{,}\) \(z \geq 1\text{;}\)

  6. \(w + x + y + z = 7\text{,}\) where \(w \geq 0\text{,}\) \(x\geq 0\text{,}\) \(y\geq 0\text{,}\) \(z \geq 0\text{;}\)

  7. \(w + x + y + z = 12\text{,}\) where \(w \geq 2\text{,}\) \(x\geq 2\text{,}\) \(y\geq 2\text{,}\) \(z \geq 2\text{;}\)

  8. \(w + x + y + z = 10\text{,}\) where \(w \geq 0\text{,}\) \(x\geq 1\text{,}\) \(y\geq 2\text{,}\) \(z \geq 3\text{;}\)

  9. \(w + x + y + z = 15\text{,}\) where \(w \geq 1\text{,}\) \(x\geq 2\text{,}\) \(y\geq 3\text{,}\) \(z \geq 4\text{;}\)

  10. \(v + w + x + y + z = 15\text{,}\) where \(v \geq 1\text{,}\) \(w \geq 0\text{,}\) \(x\geq 1\text{,}\) \(y\geq 3\text{,}\) \(z \geq 3\text{.}\)
