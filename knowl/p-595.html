The Binomial theorem can be used to calculate several \(n\)th powers. For example, choosing \(y=1\text{,}\) every power of \(y\) is 1, as well, thus

\begin{align*} (x+1)^n \amp = x^n + n x^{n-1}\cdot 1 + \binom{n}{2} x^{n-2}\cdot 1^2 + \dots + n x \cdot 1^{n-1} + 1^n\\ \amp = x^n + n x^{n-1} + \binom{n}{2} x^{n-2} + \dots + n x + 1 = \sum_{k=0}^n \binom{n}{k} x^k\text{.} \end{align*} in-context
