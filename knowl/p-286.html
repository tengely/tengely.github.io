The number of \(k\)-element ordered subsets is \(\frac{n!}{(n-k)!}\) by TheoremÂ 2.5.2. To count the number of unordered \(k\)-element subsets we group together those \(k\)-element subsets, which differ only in their order. That is, every group contains different orderings of the same \(k\) elements. Every group contains \(k!\)-many orderings, since \(k!\) is the number of possible permutations of those \(k\) elements. Therefore the number of groups (and the number of \(k\)-element subsets) is

\begin{equation*} \frac{\frac{n!}{(n-k)!}}{k!} = \frac{n!}{k! \cdot (n-k)!} = \binom{n}{k}\text{.} \end{equation*} in-context
