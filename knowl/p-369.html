An urn contains \(n\) numbered balls. How many ways can we choose \(k\) balls out of the urn if

  1. \(n = 9\text{,}\) \(k=3\text{,}\) with repetition, the order does not count;

  2. \(n = 3\text{,}\) \(k=9\text{,}\) with repetition, the order does not count;

  3. \(n = 10\text{,}\) \(k=5\text{,}\) without repetition, the order counts;

  4. \(n = 5\text{,}\) \(k=10\text{,}\) without repetition, the order counts;

  5. \(n = 45\text{,}\) \(k=6\text{,}\) without repetition, the order does not count;

  6. \(n = 6\text{,}\) \(k=45\text{,}\) without repetition, the order does not count;

  7. \(n = 100\text{,}\) \(k=10\text{,}\) with repetition, the order counts;

  8. \(n = 10\text{,}\) \(k=100\text{,}\) with repetition, the order counts.
