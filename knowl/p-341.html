We are not interested in all the solutions, we are only interested in the number of positive integer solutions. We can of course list them without any problems. First, we look for solutions \(x \geq y \geq z\text{,}\) then we find all the solutions. Now, \(z\) cannot be bigger than 2, because that would mean \(7 = x+ y + z \geq 3z \geq 3\cdot 3 = 9\) a contradiction. Thus \(z=1\) or \(z=2\text{.}\) If \(z = 2\text{,}\) then \(x\geq y \geq 2\text{,}\) as well. Now, if \(y \geq 3\text{,}\) then \(x \geq 3\text{,}\) and \(7 = x + y + z \geq 2y + z \geq 2\cdot 3 + 2 = 8\) is a contradiction. Thus if \(z=2\) then \(y=2\text{,}\) and then \(x= 3\) from (2.7.1). If \(z=1\text{,}\) then \(y\) cannot be bigger than 3 (then \(x\geq y \geq 4\text{,}\) and thus \(7 = x+y+z \geq 2y + z \geq 2 \cdot 4 + 1 = 9\) a contradiction). If \(y = 3\) then \(x=3\text{,}\) if \(y = 2\) then \(x=4\text{,}\) and if \(y = 1\) then \(x=5\text{.}\) That is, the positive integer solutions of (2.7.1) are
