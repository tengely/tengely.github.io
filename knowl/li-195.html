ExerciseÂ 3.1.14 We prove the statement by induction. If \(n=1\text{,}\) then the 1-digit integer \(a_1=2\) is divisible by 2. Therefore the statement is true for \(n=1\text{.}\) It is not difficult to deal with the case \(n=2\text{.}\) There are only four possible integers

\begin{equation*} a_1a_2\in\halmaz{11,12,21,22}\text{.} \end{equation*}

It is easy to see that \(2^2\) divides 12. Assume that the statement is true for some \(1\leq k\in\mathbb{N}\text{,}\) that is, there exists a \(k\)-digit integer \(a_1a_2\ldots a_k\) which is a multiple of \(2^k\text{.}\) Let us consider the statement for \(k+1\text{.}\) By induction hypothesis we have

\begin{equation*} a_1a_2\ldots a_k=2^k\cdot A\text{.} \end{equation*}

We claim that either

\begin{equation*} 10^{k}+a_1a_2\ldots a_k=1a_1a_2\ldots a_k \end{equation*}

or

\begin{equation*} 2\cdot 10^{k}+a_1a_2\ldots a_k=2a_1a_2\ldots a_k \end{equation*}

is a multiple of \(2^{k+1}\text{.}\) We can rewrite the above integers as follows

\begin{align*} \amp 10^{k}+a_1a_2\ldots a_k=10^k+2^k\cdot A=2^k(5^k+A),\\ \amp 2\cdot 10^{k}+a_1a_2\ldots a_k=2\cdot 10^{k}+2^k\cdot A=2^k(2\cdot 5^k+A)\text{.} \end{align*}

If \(A\) is odd, then \(5^k+A\) is even. In this case

\begin{equation*} 1a_1a_2\ldots a_k \end{equation*}

is an integer having \(k+1\) digits and it is divisible by \(2^{k+1}\text{.}\) If \(A\) is even, then \(2\cdot 5^k+A\) is even. That is,

\begin{equation*} 2a_1a_2\ldots a_k \end{equation*}

is a \((k+1)\)-digit number which is a multiple of \(2^{k+1}\text{.}\)
