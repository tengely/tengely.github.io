Similarly, as \(16 = 2^4\text{,}\) every base 16 digit can be rewritten easily to four base 2 digits:

\begin{align*} 0_{16} \amp = 0000_2, \amp 1_{16} \amp = 0001_2,\\ 2_{16} \amp = 0010_2, \amp 3_{16} \amp = 0011_2,\\ 4_{16} \amp = 0100_2, \amp 5_{16} \amp = 0101_2,\\ 6_{16} \amp = 0110_2, \amp 7_{16} \amp = 0111_2,\\ 8_{16} \amp = 1000_2, \amp 9_{16} \amp = 1001_2,\\ A_{16} \amp = 1010_2, \amp B_{16} \amp = 1011_2,\\ C_{16} \amp = 1100_2, \amp D_{16} \amp = 1101_2,\\ E_{16} \amp = 1110_2, \amp F_{16} \amp = 1111_2\text{.} \end{align*} in-context
