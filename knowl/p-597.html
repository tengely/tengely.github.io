Alternatively, we can substitute \(-y\) instead of \(y\) in the Binomial theorem, obtaining

\begin{align*} (x-y)^n \amp = x^n + n x^{n-1}\cdot (-y) + \binom{n}{2} x^{n-2}\cdot (-y)^2 + \dots + n x \cdot (-y)^{n-1} + (-y)^n\\ \amp = x^n - n x^{n-1} y + \binom{n}{2} x^{n-2} y^2 - \dots + (-1)^{n-1} n x y^{n-1} + (-1)^n y^n\\ \amp = \sum_{k=0}^n (-1)^k \binom{n}{k} x^{n-k}y^{k}\text{.} \end{align*} in-context
